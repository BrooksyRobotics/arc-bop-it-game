# arc-bop-it-game
Recreate the Head Bop-It Game to match your head movement with the signal given: Twist, Bob and Roll.

https://youtube.com/shorts/ngLXlH33Bwg

## Components: Wiring
- LED Matrix MAX7219: VCC->5V    GND->GND  DIN->11  CS->10   CLK->13
- Passive Buzzer:  (+)->Pin 8  (-)->GND
- Gyroscope/Accelerometer GY-521 module: VCC->3.3V  GND->GND  SDA->A4  SCL->A5  AD0->GND

DEPENDENCIES  (Library Manager)
- LedControl  by Eberhard Fahle
- MPU6050     by Electronic Cats
